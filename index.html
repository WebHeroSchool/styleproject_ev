Селекторы

CSS - формальный язык для описания языка разметки. Это описание стилей того, как страница будет выглядеть. Селектор - это писание того элемента, к которому мы зотим обратиться.

Базовый синтаксис

Основное понятие селектор - это некоторое имя стиля, для которого добавляются параметры форматирования. В качестве селектора выступают теги, классы и идентификаторы.

Правила применения стилей

Для селектора допускается добавлять каждое стилевое свойство и его значение по отдельности. Приоритет имеет значение, указанное в коде ниже.
